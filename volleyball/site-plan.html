Overview

Purpose

Create a dynamic site that helps volleyball fans to find the latest stats of their favorite teams and players. As well as providing interesting information regarding the sport.

Audience

Volleyball enthusiasts and beginners who want to learn more about the sport.

Dynamic elements

Use of Java to create a responsive menu, which hides and appears depending on the screen size and when user clicks on it. Responsive gallery for pictures of champions, when user clicks on the image of team, will be responsive and will create a bigger image in good resolution. Use of Java to create a dynamic display of different players card with information about them as done in week 06.
